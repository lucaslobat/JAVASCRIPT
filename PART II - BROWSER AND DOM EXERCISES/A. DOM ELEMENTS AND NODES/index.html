Recorriendo Parent/Child/Sibling

Node.parentElement

La propiedad de solo lectura parentElement de la interfaz Node, retorna el Elemento padre de un nodo, o null si el nodo no tiene padre, o si el padre no es un Elemento.

Element.children

La propiedad de solo lectura children retorna una HTMLCollection, la cual contiene todos los elementos hijos del Element sobre el cual esta ha sido llamada.

Element.nextElementSibling / Element.previousSibling

La propiedad de solo lectura Element.nextElementSibling / Element.previousElementSiblingretorna el nodo precedente/antecedente al nodo especificado en la lista de hijos del padre, o retorna null si el elemento especificado es el último hijo del elemento padre

Append & AppendChild

Document.createElement()

En un documento HTML, el método document.createElement() crea un elemento HTML con el nombre de etiqueta especificado, o una HTMLUnknownElement si el nombre de la etiqueta no es reconocido

Element.append()

El método Element.append() inserta un conjunto de objetos Node u objetosDOMString después del último Elemento hijo.

Element.prepend()

El método Element.prepend() añade un conjunto de objetos Node u objetos DOMString antes del primer hijo del Elemento especificado. Los Objetos DomString son añadidos como nodos Text
