DOM Events

Un evento es una acción se dispara en respuesta de una interacción del usuario como hacer click en el ratón,presionar una tecla o pueden ser generadas por API's para representar el progreso de una tarea asíncrona. También puede ser disparado un evento al llamar el método HTMLElement.click(), o definiendo el evento para luego mandarlo a un objetivo usando EventTarget.dispatchEvent().
